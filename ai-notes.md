## Agent 2 Agent (multi-agent communication) via A2A

MCP client talks to MCP server which internally talks to acctual implementation
Agent 2 Agent : communicates between agent client and agent server via AGENT card, skills, capabilities and Agent executors ... works on TASK (Submit) via messages, send final response as artifact 
life cycle of TASK ( start, working, fasiled, cancelled)
